# Домашнее задание №17 - Умножение матриц (2 балла)

## Хранение и представление матриц

Каждая матрица размера `X` на `Y` представляется в вводе и выводе следующим образом:
```
e[1,1] e[1,2] ... e[1,y]
e[2,1] e[2,2] ... e[2,y]
...
...
e[x,1] e[x,2] ... e[x,y]
```

Каждая матрица должна храниться в динамической памяти и представляться в программе в виде
двумерного VLA-массива.

## Условие

Прочитать с клавиатуры три числа `N`, `M` и `K` типа `size_t`. Затем прочитать матрицу `A` размера
`N` на `M` и матрицу `B` размера `M` на `K`. Вывести произведение матриц `A` и `B`.

Операция умножения матриц должна быть выделена в функцию, принимающую VLA-массивы и возвращающую
VLA-массив.

## Пример

### Входные данные

```
2 2 2
1 2
3 4
5 6
7 8
```

### Выходные данные

```
19 22
43 50

```
